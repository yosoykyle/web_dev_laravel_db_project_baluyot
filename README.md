## Project Overview

This project is a **Student Management System** built with **Laravel 10** and **Bootstrap 5**. It demonstrates how to connect a Laravel application to a **MySQL** database, perform migrations, create models and controllers, and display data using Blade templates.

## Key Features

-   **Database Integration**: Connects to a MySQL database using Laravel's Eloquent ORM.
-   **Migrations**: Uses migration files to define the database schema (Table: `students`).
-   **Model & Controller**: Implements MVC architecture with a `Student` model and `StudentController`.
-   **Routing**: Defines routes for listing all students and viewing individual student details.
-   **Bootstrap UI**: detailed, responsive user interface using Bootstrap 5 (via Vite).
-   **Dynamic Views**: Displays student data dynamically from the database.

## Technologies Used

-   **Framework**: Laravel
-   **Language**: PHP
-   **Database**: MySQL
-   **Frontend**: HTML, CSS, Bootstrap 5
-   **Tooling**: Vite, Composer, Node.js
